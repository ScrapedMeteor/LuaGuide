# Math

operators
- \+ addition
- \- minus
- \* multiply
- / divide
- ^ power
- % modulus

```lua
-- example

local Age = 18
local NewAge = Age + 5
print(NewAge)
```

# Math Comparsion

- == equality
- < less than
- \> greater than
- <= less than or equal to
- \>= greater than or equal to
- ~= inequality

```lua
--example 
local MarriageAge = 23
if MarriageAge < 23 then
    print("no marriage :(")
```
